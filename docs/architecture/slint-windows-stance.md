# Slint Windows Stance

Windows is not a supported Slint productization target for this branch.

Shared Rust may keep Windows-compatible code where it already exists, but Slint product acceptance is limited to:

- Linux desktop
- macOS desktop
- Android evaluation
- iOS simulator/evaluation

Do not add Windows-specific Slint UI, packaging, or visual-fidelity gates unless a future bd decision changes the target matrix.
